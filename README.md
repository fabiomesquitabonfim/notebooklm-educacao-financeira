# Educação Financeira para Iniciantes com NotebookLM

## Contexto

Este projeto foi desenvolvido como parte de um desafio da DIO com o objetivo de explorar o uso do NotebookLM como ferramenta de aprendizagem ativa e organização do conhecimento.

O tema escolhido foi Educação Financeira para Iniciantes, por ser um assunto fundamental para a construção de uma vida financeira mais organizada e sustentável.

## Objetivos

* Compreender os principais conceitos da educação financeira.
* Aprender sobre planejamento financeiro pessoal.
* Entender a importância da reserva de emergência.
* Identificar boas práticas para organização do orçamento.
* Utilizar o NotebookLM para sintetizar e estruturar conhecimentos provenientes de diferentes fontes.
